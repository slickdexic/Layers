# Keyboard Shortcuts

Master keyboard shortcuts for faster, more efficient editing.

---

## Quick Reference Card

Press **`Shift+?`** at any time to display the keyboard shortcuts help dialog.

---

## Tool Shortcuts

| Key | Tool | Description |
|-----|------|-------------|
| `V` | Pointer | Select, move, resize, rotate layers |
| `Z` | Zoom | Zoom and pan the canvas |
| `T` | Text | Add single-line text labels |
| `X` | Text Box | Add multi-line text with container |
| `P` | Pen | Freehand path drawing |
| `R` | Rectangle | Draw rectangles |
| `C` | Circle | Draw circles |
| `E` | Ellipse | Draw ellipses |
| `G` | Polygon | Draw polygons |
| `S` | Star | Draw stars |
| `A` | Arrow | Draw arrows |
| `L` | Line | Draw straight lines |
| `B` | Blur | Blur/redact areas |

## Edit Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Ctrl+Shift+Z` | Redo (alternative) |
| `Ctrl+C` | Copy selected layers |
| `Ctrl+X` | Cut selected layers |
| `Ctrl+V` | Paste layers |
| `Ctrl+D` | Duplicate selected layers |
| `Delete` | Delete selected layers |
| `Backspace` | Delete selected layers (alternative) |
| `Ctrl+A` | Select all layers |
| `Escape` | Deselect all / Cancel current operation |

## Drawing Modifiers

These modifiers work while drawing shapes:

| Modifier | Effect |
|----------|--------|
| `Shift` | Constrain proportions (square, circle) |
| `Alt` | Draw from center |
| `Shift+Alt` | Both: constrain proportions from center |

## Platform Differences

### macOS Users
Replace `Ctrl` with `Cmd (⌘)` for all shortcuts:

| Windows/Linux | macOS |
|---------------|-------|
| `Ctrl+Z` | `Cmd+Z` |
| `Ctrl+S` | `Cmd+S` |
| `Ctrl+C` | `Cmd+C` |
| etc. | etc. |

---

## Shortcut Conflicts

If shortcuts conflict with browser or MediaWiki shortcuts:

1. Click inside the editor canvas first to focus it
2. Editor shortcuts take precedence when focused
3. Click outside the editor to return to normal browser shortcuts

## Tips for Efficiency

### Learn the Tool Shortcuts First
The single-key tool shortcuts (`V`, `T`, `R`, `A`, etc.) are the most valuable to memorize. They let you switch tools instantly without reaching for the mouse.

### Use Both Hands
- Left hand on keyboard for shortcuts
- Right hand on mouse for drawing/selecting

### Common Workflows

**Quick Annotation:**
1. `T` → Click → Type → Enter → `V`

**Shape with Arrow:**
1. `R` → Draw rectangle → `A` → Draw arrow pointing to it → `V`

**Duplicate and Modify:**
1. Select layer → `Ctrl+D` → Drag copy to new position → Modify
